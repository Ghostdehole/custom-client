元旦更新日志
本次更新聚焦于安全性增强、移动端体验优化与功能精简，以下是主要变更：

🛠️ 功能修复
修复打印驱动问题
解决部分系统下远程打印失败或卡顿的问题。
📱 前端体验优化
适配移动终端
重构前端页面布局
🔒 安全与策略增强
更改密码策略

自定义密码复杂度要求，更改错误次数。
工作流中服务器信息加密

敏感配置在工作流中自动加密。
跨站请求伪造（CSRF）防护

启用严格 CSRF 验证，防止恶意站点发起非授权操作。
根据环境变量指定域名访问

支持通过 DOMAIN 或 ALLOWED_HOSTS 环境变量限定合法访问域名，提升部署安全性。
👁️ 界面精简与隐藏功能
为满足特定场景下的简洁性与隐蔽性需求

无固定密码 + 隐藏连接管理器

支持不设置固定连接密码，并在客户端完全隐藏连接管理入口。
隐藏会话工具栏的聊天功能

远程会话期间不显示顶部工具栏及聊天窗口，减少视觉干扰。
移除退出按钮

禁用客户端内的“退出”或“关闭”按钮，防止误操作。
移除客户端有密码警告

不再显示“已设置密码”等提示信息。
禁用设置菜单

锁定客户端设置入口，用户无法修改本地配置。
移除密码框

无需显示密码。
禁用隐私模式快捷键

阻止用户通过快捷键退出隐私浏览模式。
